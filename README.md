## Project Overview
This project focuses on analyzing Netflix content data to uncover insights about movies and TV shows available on the platform. The dataset was cleaned, transformed, and analyzed using Python and Pandas, followed by the creation of an interactive dashboard using Power BI.

The goal of this project is to understand content distribution, genre popularity, country-wise production, and growth trends in Netflix's catalog.

This project demonstrates key data analyst skills including data cleaning, exploratory data analysis (EDA), feature engineering, and dashboard creation.

## Dataset

The dataset used for this analysis was obtained from Kaggle and contains information about Netflix movies and TV shows.

Key attributes in the dataset include:

- Title  
- Type (Movie / TV Show)  
- Genre  
- Country  
- Release Year  
- Date Added  
- Rating  
- Duration  

---

## Data Cleaning and Preprocessing

Several preprocessing steps were performed to ensure the dataset was suitable for analysis:

- Checked dataset structure using Pandas
- Handled missing values
- Removed duplicates
- Converted date columns to datetime format
- Split multi-value columns (Genre, Country)
- Used explode() to separate multiple genres
- Applied groupby() operations for aggregation
- Performed feature engineering to create new useful columns

These steps improved data quality and enabled accurate analysis.

## Exploratory Data Analysis

The dataset was analyzed to identify patterns and trends in Netflix content.

Key analysis performed:

- Distribution of Movies vs TV Shows
- Genre popularity analysis
- Country-wise content distribution
- Rating category analysis
- Average duration of movies by genre
- Content growth trend over the years

---

## Dashboard Insights

### Movies vs TV Shows
Movies make up the majority of Netflix's content library compared to TV Shows.

### Genre Distribution
Drama, International Movies, and Comedy are among the most common genres on Netflix.

### Country Contribution
The United States contributes the largest number of titles, followed by countries such as India and the United Kingdom.

### Content Growth
Netflix content has grown significantly after 2016, reflecting rapid expansion of the platform.

### Duration Analysis
Different genres show variation in average movie duration.
